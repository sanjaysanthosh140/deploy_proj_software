# Mirror Edge Effect

Components should have **glass-like reflective edges** similar to polished crystal.

Design details:

- bright edge highlights
- subtle corner reflections
- thin glowing borders
- light reflections along edges

Edges should look like **real glass catching light**.

---

# Cards (Service Components)

Cards should look like **floating glass tiles**.

Features:

- glass background
- frosted blur
- reflective border edges
- smooth rounded corners
- soft drop shadow

Hover interaction:

- card slightly lifts
- shadow becomes deeper
- edge reflections become brighter

---

# UI Components Using This Style

Apply the same glass design to:

- service cards
- navigation bar
- feature sections
- buttons
- modal windows
- content panels
- pricing cards

All components should feel like **part of one consistent glass UI system**.

---

# Color System

Background  
`#ffffff`

Glass surface  
`rgba(255,255,255,0.6)`

Border highlight  
`rgba(255,255,255,0.8)`

Primary text  
`#1a1a1a`

Secondary text  
`#6b7280`

Accent metallic tone  
`#c0c0c0`

---

# Lighting & Depth

Use subtle lighting effects:

- soft shadows under components
- inner glass reflections
- faint top light highlights
- gentle gradient lighting

Avoid dark backgrounds or heavy contrast.

---

# Final Look

The UI should feel like a **premium glass interface for a modern web application**, where every component appears like **floating polished glass with mirror edges on a clean white background**.
