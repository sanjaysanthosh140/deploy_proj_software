# Glass Mirror UI Design System

Design a **modern web application UI** using a **white background** and **glass mirror-style components**.  
The interface should feel **premium, futuristic, and minimal**, inspired by **polished glass panels and aluminium edges**.

---

# Overall Visual Style

The UI should look like **floating glass components placed on a clean white background**.

Design inspiration:
- Glassmorphism
- Polished mirror edges
- Apple-style minimal UI
- Modern architecture interfaces

The design should feel **light, spacious, and elegant**.

---

# Background Design

Use a **soft white background**, not dark.

Background palette:

Primary background  
`#ffffff`

Secondary background  
`#f5f7fa`

Soft gradient areas  
`#eef1f5`

Add subtle depth using:

- soft gradient layers
- faint radial light
- very subtle noise texture
- minimal blur effects

The background should remain **clean and minimal so glass elements stand out**.

---

# Glass Component Style

All UI components should appear like **transparent glass panels**.

Component characteristics:

- semi-transparent white surface
- frosted blur effect
- mirror-like edges
- soft shadows
- rounded corners
